Descripción del programa: Este reto debe de generar un grafo completo dado
N numero de nodos. El programa dibuja mediante un algoritmo de Bresenham 
las aristas que conectan cada par de nodos, punto a punto.
Nota: El grafo que se genera es simetrico y la posicion de cada nodo se
define mediante la diferencia entre nodos, definida como 360/#DeNodos.

El proyecto contiene dos archivos de java, por un lado la interfaz que
permite la facil interaccion del usuario con las variables de numero de
nodos, tamaño del grafo y color, ademas de algunas otras opciones.
El archivo de retoLineasConExtra, genera la interfaz basica donde solo se
requiere el numero de nodos por parte del usuario. La interfaz es mas
minimalista en el momento de dibujar el grafo.
